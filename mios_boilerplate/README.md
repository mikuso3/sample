# Mio's Boilerplate


## 必要なファイル
npm install nodemon（監視）<br>
npm install ejs-cli（ejsのコンパイル）<br>
npm install node-sass（スタイルのコンパイル）
npm install node-sass-globbing（importを一纏めにできる）

## ejsのビルド
'npm run ejs'

## スタイルを監視
'npm run watch:style'

## 全てを実行
'npm run all'
