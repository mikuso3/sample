# Mio's Boilerplate

## Using Package
npm i nodemon（監視）<br>
npm i ejs-cli（ejsのコンパイル）<br>
npm i node-sass（スタイルのコンパイル）
npm i node-sass-globbing（importを一纏めにできる）

## Build of ejs
'npm run ejs'

## Watch All
'npm run watch'

## Add hash
'npm run hash'

# Copy to folder of public
'npm run copy'
